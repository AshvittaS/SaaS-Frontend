# SaaS-Frontend

A modern SaaS home page built with pure HTML and CSS. This is a beginner-friendly project featuring a clean, responsive design with a red and black color theme.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Modern Layout** - Clean and professional design
- **Red & Black Theme** - Bold color scheme with red accents on black background
- **No Dependencies** - Pure HTML and CSS, no JavaScript or frameworks required
- **Beginner Friendly** - Simple code structure, easy to understand and modify

## Sections

1. **Navigation Bar** - Sticky header with logo, menu links, and action buttons
2. **Hero Section** - Main landing area with headline, description, and call-to-action buttons
3. **Features Section** - Showcase of 6 key features with icons
4. **Pricing Section** - Three pricing tiers (Starter, Professional, Enterprise)
5. **Call-to-Action Section** - Final conversion section
6. **Footer** - Links and company information

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No installation or build process needed.

## File Structure

```
SaaS-Frontend/
├── index.html      # Main HTML file
├── styles.css      # All styling and layout
└── README.md       # This file
```

## Customization

The design is easy to customize:

- **Colors**: Edit the color values in `styles.css` (currently using red #dc2626 and black #000)
- **Content**: Modify the text in `index.html`
- **Layout**: Adjust the grid and flexbox properties in `styles.css`

## Browser Support

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge
